# Connors Heating Project

## Project Documentation

This repository hosts the code and documentation for the Connors Heating project. This project aims to provide efficient heating solutions to residential and commercial buildings.

## Setup Instructions

To set up the project on your local machine, follow these steps:

1. **Clone the repository**:
   ```bash
   git clone https://github.com/connorsheating/Connorsheating.git
   ```

2. **Navigate to the project directory**:
   ```bash
   cd Connorsheating
   ```

3. **Install the necessary dependencies**:
   ```bash
   npm install
   ```

4. **Run the application**:
   ```bash
   npm start
   ```

5. **Access the application** at `http://localhost:3000`.
